# Notes App
This is a notes app made with Node, Express and Mysql.

# Steps to Setup
## 1. Install dependencies
```
npm install
```
## 2. Configure Enviroment Variables
Add your enviroment variables: 
```
// APP
PORT=
NODE_ENV=

// DATABASE
DB_USER=
DB_PASSWORD=
DB_HOST=
DB_NAME=
```
## 3. Configure database
You can use the schema included on **src/lib**.

## 4. Run 
```
# On development mode
npm run dev

# On production mode
npm run start
```
